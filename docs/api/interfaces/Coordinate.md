# Interface: Coordinate

A simple geographic coordinate expressed as latitude and longitude in
decimal degrees (WGS-84).

## Example

```ts
const coord: Coordinate = { lat: 51.5, lng: -0.1 };
```

## Properties

### lat

> **lat**: `number`

Latitude in decimal degrees. Range: −90 to 90.
